# Survival History of AS6809

One reason this project exists today is that the source code survived across multiple generations of computers and storage media.

Approximate migration path:

* Around 1988\
  FM-new7 (original BASIC version)\
  → cassette tape\
  → PC-8801 (home)

* Around 1989\
  PC-8801mkIISR (home)\
  → floppy disk\
  → PC-9801 (university laboratory)

* Around 1990\
  PC-9801 (university laboratory)\
  → floppy disk\
  → X68000 (home)

* Around 1996\
  X68000 (home)\
  → MO disk\
  → PC/AT compatible machine (home)

* Around 2015\
  PC/AT machine (home)\
  → Internet transfer\
  → Google Drive

Because of these migrations, the source code remains available today.
